Mon Expérience de Stage

Durant mon BTS SIO, j’ai eu l’opportunité d’effectuer un stage en entreprise, ce qui m’a permis d’acquérir des compétences techniques et professionnelles précieuses.

📌 Informations sur mon stage

  • Entreprise : Société Florquin Couverture
  • Période : 19 mai 2025 ~ 26 juin 2025
  • Mission principale : Créer un fichier client numérique

🚀 Compétences acquises

  • Création et organisation d’un fichier client sous Excel
  • Développement en Python d’un script pour importer les données Excel dans une base de données
  • Création d’une interface utilisateur simple pour interagir avec les données
  • Autonomie dans la gestion de projet et résolution de problèmes techniques
  • Communication et recueil des besoins auprès des responsables

📝 Bilan du stage

Ce stage m’a offert une expérience de gestion de projet informatique. J’ai commencé par concevoir un fichier client sous Excel, puis développé un script Python permettant de transférer ces données dans une base de données avec une interface utilisateur simple. Cette autonomie m’a permis de renforcer mes compétences en programmation, et analyse des besoins, tout en m’adaptant à un environnement sans encadrement technique direct.
